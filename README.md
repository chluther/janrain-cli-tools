jcli
====

Janrain CLI.

## Install from source

```
git clone git@github.com:xwp/janrain-cli-tools.git jcli
cd jcli
composer install
```

Done.

If you want to build the phar file:

```
box build
```

and you can move the file to your OS `PATH`:

```
mv jcli.phar /usr/local/bin/jcli
```


## Commands

```
./bin/jcli
```
